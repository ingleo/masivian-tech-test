API REST para manejo de arbol binario
Método HTTP URL Body Acción
POST https://leo-masiv-tst.herokuapp.com/v1/nodos {secuencia: 70,84,85} Crea los nodos a un arbol, sino existe crea el arbol
GET https://leo-masiv-tst.herokuapp.com/v1/nodos/ancestro/?nuno=29&ndos=55 N/A Obtiene el ancestro común mas cercano de dos nodos
GET https://leo-masiv-tst.herokuapp.com/v1/arbol N/A Obtiene el arbol binario actual
DELETE https://leo-masiv-tst.herokuapp.com/v1/arbol N/A Borra el arbol binario actual
